HyperText Markup Language (HTML)

What is a Website?

A website is a collection of related web pages that are connected under a single domain name (like www.google.com). It is stored on a web server and can be accessed through the internet using a browser.

EXAMPLE: Google, Wikipedia, all are websites.

What is a WebPage?

A WebPage is a single document on the internet. It is like one page of a book—part of the full website. It is written in a coding language (usually HTML) and displayed through a web browser.

EXAMPLE: Wikipedia HTML page
(web page of the Wikipedia website)

What is HTML?

HTML stands for HyperText Markup Language. It is a standard language used to create and design web pages.

Think of HTML as the bones of a webpage—it gives structure, just like a skeleton gives structure to the body. It is the code that is used to structure a web page and its content.

It tells the browser what to show—headings, text, images, links etc.

HTML = Structure

CSS = Style

JavaScript = Behavior

Angle Brackets

Angle brackets are used in HTML to define tags, which are the building blocks of a webpage.

These tags tell the browser what type of content is being written and how to display it.

Tags written in angle brackets help separate HTML instructions from the actual text content, allowing the browser to properly render headings, paragraphs, links, images, and more.

Without <>, the browser would not recognize the tags as HTML code and wouldn't display the content correctly.

What are HTML tags?

They are the building blocks of an HTML document. They are special code written inside angle brackets that tell the web browser how to display content on a webpage.
