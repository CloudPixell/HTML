What is a Website?

A website is a collection of related web pages under a single domain name (like www.google.com), stored on a web server and accessible through a browser.

Example: Google, Wikipedia, Instagram â€” all are websites.

What is a Web Page?

A web page is a single document on the internet, like one page of a book. It's part of a website and is written using HTML. A web browser reads this code and displays it properly.

Example: Wikipedia HTML Page is a web page of the Wikipedia website.

What is HTML?

HTML stands for HyperText Markup Language. It provides the structure of a web page, like the skeleton of a human body.

It helps browsers display content such as headings, text, images, and links properly.

HTML = Structure
CSS = Style
JavaScript = Behavior

Angle Brackets in HTML

Angle brackets < > are used to define HTML tags. Tags tell the browser what kind of content to display (like headings, images, links, etc.).

Without these brackets, the browser would not know you're writing HTML code.

What are HTML Tags?

HTML tags are the building blocks of any HTML document. They're enclosed in angle brackets and tell the browser how to show content.

Example:

<p>This is a paragraph.</p>
